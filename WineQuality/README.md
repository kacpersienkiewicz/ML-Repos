Poking around the famous [Wine Quality](https://archive.ics.uci.edu/dataset/186/wine+quality) dataset to create a model to classify red and white wine.

I use cross validation to test K Nearest Neighbors, Random Forest, and Logistic Regression, and while all three models scored incredibly well (likely because the data is especially well suited to this sort of classification), the Random Forest performed the best. I then used RandomizedSearchCV to try to figure out the best parameters for the Random Forest. In the end, the model was incredibly good (basically 100%), which concerned me, since models are typically not doing that well unless they're overfitted, or one of the features are either the same as the target feature or derived from it (or vice versa).

# Background
The premise is that your client is a credit card company, who wants to predict defaults on payments, and who could be helped with credit counseling, with the hope of them not defaulting.
# Program
The program will be a Jupyter Notebook that showcases several Machine Learning algorithms (GLMs like logistic and linear regression, Decision Trees, ensemble models like Random Forests), evaluation methods like ROC AUC, and feature selection methods.
